# 💬 WhatsApp Floating Bubbles

Tres variantes de botón flotante de WhatsApp listas para copiar y pegar en cualquier sitio web. Sin dependencias, sin frameworks, solo HTML puro.

## ¿Qué contiene este repositorio?

```
whatsapp-bubbles/
└── index.html   ← las 3 burbujas con demo y comentarios
```

## Las 3 variantes

| Variante | Cuándo usarla |
|---|---|
| **Opción 1 — Clásica simple** | Cualquier sitio. Mínima y universal. Solo el ícono. |
| **Opción 2 — Con tooltip** | Páginas de servicios o ventas. Añade texto persuasivo. |
| **Opción 3 — Mini chat popup** | Mayor conversión. Muestra un mini chat antes de abrir WhatsApp. |

## Cómo usar el código

1. Abre `index.html`
2. Encuentra el bloque que necesitas (busca `COPIA DESDE AQUÍ`)
3. Copia todo hasta `FIN BURBUJA X`
4. Pégalo justo antes del `</body>` de tu página
5. Cambia el número de teléfono por el tuyo

## Personalización rápida

### Cambiar el número de teléfono
Busca `573155023405` y reemplaza por tu número en formato internacional (sin `+` ni espacios).

**Ejemplo:** Si tu número es `+57 300 123 4567` → escribe `573001234567`

### Cambiar el mensaje predefinido
Busca `?text=` y edita el texto después. Los espacios se escriben como `%20`.

### Cambiar el texto del tooltip (Opción 2)
Busca `¿Tienes preguntas? ¡Escríbenos!` y cambia por tu propio mensaje.

### Cambiar nombre y avatar (Opción 3)
- Nombre: busca `Soporte en línea` y cambia por el tuyo
- Avatar (letra): busca `>S<` y cambia la `S` por tu inicial
